# Creating organizations

An organization is the top-level workspace in Platform. It groups your namespaces, teammates, API keys, and settings.

## Create your first organization

1. Sign in to Platform.
2. Open the **Team Dashboard** entry from the main app navigation.
3. If you do not belong to an organization yet, Platform shows a **Create organization** screen.
4. Enter an organization name.
5. Submit the form to create the workspace.

After creation, Platform routes you directly into namespace onboarding for that organization.

## What happens next

- The new organization becomes your active workspace.
- Platform sends you to `/dashboard/<org>/new-namespace`.
- From there, you import an ENS name and finish namespace setup.

## Managing the organization later

Admins can return to **Org Management** to:

- invite members by email,
- review or change organization roles, and
- create organization-level API keys.

If you already belong to one or more organizations, Platform redirects you into the first available workspace instead of showing the empty-state create screen.
