# Exploring accounts, contracts, and ENS names

import AppUrl from '@site/src/components/AppUrl';

Explore mode is the fastest way to try Enscribe if you do not have an ENS name or a contract to name yet.

## Try it now

- Open a live Enscribe-named contract:
  <AppUrl path="explore/1/0x9b95a964db0ab154a587cc667b4964bd5dd8de81" label="Explore an Enscribe contract on Ethereum" />
- Open a known protocol contract:
  <AppUrl path="explore/1/0x66a9893cC07D91D95644AEDD05D03f95e1dBA8Af" label="Explore a Uniswap contract on Ethereum" />
- You can also paste any address or ENS name in the app search bar.

## Universal search

Search works without wallet connection. Paste a `0x...` address or type an ENS name.

![Universal search bar](../../blog/2025-06-10-explore-and-search-address/searchbar.png)

Search supports:
- Ethereum addresses (`0x...`)
- ENS names
- Chain-aware routing to the correct contract/account page

## Direct URL format

You can share any contract/account with:

```text
https://app.enscribe.xyz/explore/<chainId>/<address>
```

This format is useful for support docs, runbooks, and team handoffs.

## What you can inspect on contract pages

- Primary ENS name and expiry state
- Verification status across Sourcify, Etherscan, and Blockscout
- Links to verified source code
- Names resolving to the contract
- Names owned or managed by the contract

![Contract details view](../../blog/2025-06-10-explore-and-search-address/contract.png)

## What you can inspect on account pages

- Primary ENS name and expiry state
- Names resolving to the account
- Names owned or managed by the account

![Account details view](../../blog/2025-06-10-explore-and-search-address/account.png)

## Chain selector behavior

When no wallet is connected:
- You can choose any supported chain
- Data loads for that selected chain only

![Chain selector with wallet disconnected](../../blog/2025-06-10-explore-and-search-address/chain-selector.png)

When a wallet is connected:
- The selected chain follows your wallet network
- Changing wallet network updates the page to the same address on that chain

![Chain selector with wallet connected](../../blog/2025-06-10-explore-and-search-address/wallet-selector.png)

## Supported networks

- Ethereum Mainnet
- Ethereum Sepolia
- Base Mainnet
- Base Sepolia
- Linea Mainnet
- Linea Sepolia

You can use Explore as a zero-setup entry point, then move to naming once you are ready.
