# Overview

import DocCardList from '@theme/DocCardList';
import {useCurrentSidebarCategory} from '@docusaurus/theme-common';

Manager mode is the control model Enscribe uses for a namespace.

When you onboard a namespace, you choose how manager access should be held and how write actions should flow. This choice affects who controls the ENS name and how your team approves updates such as subname creation and record changes.

## Available manager modes

Enscribe currently shows three manager modes:

- **Enscribe** for teams that want a more direct managed setup.
- **Safe** for teams that want multisig approvals around namespace writes.
- **Smart session** for a future mode that is not available yet.
