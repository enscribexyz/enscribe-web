# What is Enscribe?

Enscribe is infrastructure for smart contract identity on ENS.

It helps teams:
- Name new contracts at deployment time
- Name existing contracts after deployment
- Add and inspect contract metadata and verification context
- Explore contract and account identity across multiple chains

## Why Enscribe exists

In many apps and wallets, contracts still appear as raw hex addresses. That creates operational and security risk because users and teams cannot quickly tell what a contract is.

Enscribe reduces that ambiguity by making ENS-based contract naming and metadata management practical in day-to-day workflows.

## What you can do in Enscribe

1. Deploy and name in one flow
2. Name existing contracts with forward resolution and, when supported, primary names
3. Browse named contracts and accounts in Explore mode
4. Track contracts in deployment history
